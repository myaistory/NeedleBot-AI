## 2. 三大核心模块

### 2.1 极低延迟数据源 (gRPC Geyser)

**配置要求:**
- 使用 Helius 或 Jito 专用节点
- gRPC 延迟 < 100ms
- WebSocket 实时推送

**实现代码:**
```javascript
// 连接 Solana Geyser
const connection = new Connection('https://api.mainnet-beta.solana.com');
```

### 2.2 真假插针识别 (Falling Knife Detection)

**判定逻辑:**

| 条件 | 结果 | 操作 |
|------|------|------|
| 跌幅>20% + 流动性下降 | 🚫 RUG | 放弃 |
| 跌幅>15% + LP锁定正常 | ✅ 插针 | 买入 |
| 跌幅>10% + 5min占1h>50% | ⚡ 快速插针 | 立即买入 |
| 持续下跌(5m+1h+24h同向) | 💀 归零 | 禁止买入 |

### 2.3 动态Jito Tip策略

**算法:**
```
基础费用 = getRecentPrioritizationFees()
拥堵度 = HIGH (>0.001 SOL) / MEDIUM / LOW

最终小费 = 基础费用 × 拥堵系数 × 紧急系数

拥堵系数: LOW=1.0, MEDIUM=1.2, HIGH=1.5
紧急系数: normal=1.0, high=2.0, low=0.8
```

## 3. 风控系统

### 3.1 黑名单制度
- 自动记录Rug代币地址
- 记录频繁改名开发者
- 记录撤池子行为

### 3.2 仓位管理
- 最大单笔: 0.5 SOL
- 最大日交易: 10笔
- 止损线: 5%
- 止盈线: 20%

### 3.3 分批离场策略
```
盈利 20% -> 撤出本金
盈利 40% -> 撤出50%利润
盈利 60% -> 全部撤出，使用追踪止损
```
